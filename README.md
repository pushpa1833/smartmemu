# 🎉 College Digital Canteen Menu System

A web-based **Digital Canteen Menu System** that allows users to view the canteen menu online using a QR code. This system provides a smart, contactless, and paperless way for students to check food items and prices from their phones.

---

## 📌 About the Project

The **College Digital Canteen Menu System** is a responsive web application designed to replace traditional paper menus in a college canteen. By scanning a QR code, users can instantly open the digital menu on their mobile devices without installing any app.

✔ View all menu items  
✔ Check prices and descriptions  
✔ Mobile-friendly access via QR code  
✔ Paperless and efficient  

---

## 🚀 Features

- 📱 **QR Code Based Access:** Students can scan a QR code to view the menu on their phones.
- 🍔 **Digital Menu Display:** Shows food items, descriptions, and prices.
- 🖥️ **Responsive UI:** Works smoothly on desktop and mobile.
- 📂 Organized with HTML, CSS, and JavaScript.
